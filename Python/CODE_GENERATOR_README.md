# Code File Generator

A comprehensive Python-based tool that generates code files with appropriate language extensions and template code for various programming languages.

## Features

- **Support for 10 Programming Languages**: Python, JavaScript, TypeScript, Java, C++, Go, Rust, PHP, Ruby, and Swift
- **Multiple Usage Modes**: CLI mode for automation and interactive mode for guided usage
- **Template Code**: Each language includes properly formatted boilerplate code with:
  - Basic structure and syntax
  - Helpful comments
  - Example functions
  - Best practices for the language
- **Smart Features**:
  - Automatic directory creation
  - File extension handling
  - Error detection and user-friendly messages
  - Proper file naming conventions (e.g., Java class names)

## Installation

No installation required! Just ensure you have Python 3.6 or higher:

```bash
python3 --version
```

## Usage

### Interactive Mode

Run the tool without arguments for an interactive menu:

```bash
python3 code_generator.py
```

You'll be prompted to:
1. Select a programming language (1-10)
2. Enter a filename (without extension)
3. Optionally specify an output directory

### Command-Line Mode

For automation or scripting, use command-line arguments:

```bash
python3 code_generator.py -l <language> -f <filename> [-o <output_dir>]
```

**Examples:**

```bash
# Generate a Python script
python3 code_generator.py -l python -f my_script

# Generate a Java class
python3 code_generator.py -l java -f HelloWorld

# Generate TypeScript file in specific directory
python3 code_generator.py -l typescript -f app -o src/components

# Generate C++ file in nested directory (auto-creates directories)
python3 code_generator.py -l cpp -f main -o src/utils/helpers
```

### List Supported Languages

To see all supported languages and their extensions:

```bash
python3 code_generator.py --list
```

### Help

To see all available options:

```bash
python3 code_generator.py --help
```

| Java       | `.java`   | Public class, main method, example method  |

| Go         | `.go`     | Main function, package, imports            |
| Rust       | `.rs`     | Main function, example function            |
| PHP        | `.php`    | Main function, proper PHP tags             |
| Ruby       | `.rb`     | Main function, shebang, RDoc comments      |
| Swift      | `.swift`  | Main function, import Foundation           |

## Features in Detail

### Automatic Directory Creation

The tool automatically creates any necessary directories:

```bash
python3 code_generator.py -l python -f utils -o src/helpers/utilities
# Creates: src/helpers/utilities/utils.py
# Automatically creates all parent directories
```

### Smart File Extension Handling

The tool handles file extensions intelligently:

```bash
# These both create the same file: script.py
python3 code_generator.py -l python -f script
python3 code_generator.py -l python -f script.py
```

### Error Handling

The tool provides clear error messages:

- **File already exists**: Prevents accidental overwrites
- **Invalid language**: Shows list of supported languages
- **Missing arguments**: Helpful guidance on correct usage
- **Directory creation errors**: Clear error messages with details

### Java Class Name Conversion

For Java files, the tool ensures class names follow Java conventions:

```bash
# Input: hello_world or hello-world
# Output: public class HelloWorld

# Input: HelloWorld (already PascalCase)
# Output: public class HelloWorld (preserved)
```
